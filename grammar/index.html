01N3

～ことにする

决定……

毎朝走ることにしました。
02N3

～ことになる

被决定为……

来月大阪へ転勤することになりました。

～ばかり

GRAMMAR LOOKUP

语法库查询

输入中文意思、日语形式或接续关键词。

先记使用场景，再记接续形式找到 25 条语法
N5
～てください请……
＋
接续

动词て形

理解重点

礼貌提出请求。对上级可使用更礼貌的表达。

例句

もう一度言ってください。

请再说一次。
N5
～てもいい可以……
＋
接续

动词て形 + もいい

理解重点

表示许可；疑问形式用于征求许可。

例句

ここに座ってもいいですか。

可以坐这里吗？
N5
～なければならない必须……
＋
接续

动词ない形去い + ければならない

理解重点

表示义务，口语中常缩略为「なきゃ」。

例句

明日早く起きなければなりません。

明天必须早起。
N5
～たり～たりする做做……之类
＋
接续

动词た形 + り

理解重点

列举代表性动作，不表示全部。

例句

休日は本を読んだり、散歩したりします。

休息日会读书、散步之类。
N5
～から因为……
＋
接续

普通形 / 名词・形容动词 + だから

理解重点

口语中直接说明原因，语气比「ので」更主观。

例句

雨だから、家にいます。

因为下雨，所以待在家。
N4
～ようになる变得能够……
＋
接续

动词辞书形 / ない形 + ようになる

理解重点

描述能力或习惯随时间发生变化。

例句

日本語が話せるようになりました。

变得会说日语了。
N4
～ながら一边……一边……
＋
接续

动词ます形去ます + ながら

理解重点

两个动作由同一主体完成，后项是主要动作。

例句

音楽を聞きながら勉強します。

一边听音乐一边学习。
N4
～そうだ（样态）看起来要……
＋
接续

动词ます形 / 形容词词干 + そうだ

理解重点

根据眼前迹象判断，不用于一眼可知的名词性质。

例句

雨が降りそうです。

看起来要下雨。
N4
～てしまう做完；遗憾地……
＋
接续

动词て形 + しまう

理解重点

既可表示彻底完成，也可表达遗憾。口语常说「ちゃう」。

例句

宿題を忘れてしまいました。

不小心忘记作业了。
N4
～かもしれない也许、可能
＋
接续

普通形 + かもしれない

理解重点

表示不确定推测，可能性低于「でしょう」。

例句

電車が遅れるかもしれません。

电车可能会晚点。
N3
～ことにする决定……
＋
接续

动词辞书形 / ない形 + ことにする

理解重点

强调说话人主动做出的决定。

例句

毎朝走ることにしました。

我决定每天早晨跑步。
N3
～ことになる被决定为……
＋
接续

动词辞书形 / ない形 + ことになる

理解重点

表示由规则、安排或外部因素形成的结果。

例句

来月大阪へ転勤することになりました。

决定下个月调往大阪。
N3
～ばかり总是……；刚刚……
＋
接续

名词 / 动词て形 / 动词た形 + ばかり

理解重点

接续不同含义不同，要结合前项形式判断。

例句

弟はゲームばかりしています。

弟弟总是在玩游戏。
N3
～おかげで多亏……
＋
接续

普通形 + おかげで / 名词 + のおかげで

理解重点

通常连接正面结果；负面结果可带讽刺意味。

例句

先生のおかげで合格できました。

多亏老师，我通过了。
N3
～について关于……
＋
接续

名词 + について

理解重点

用于引出讨论、调查或说明的主题。

例句

日本の教育について調べています。

正在调查日本教育。
N2
～わけではない并不是……
＋
接续

普通形 + わけではない

理解重点

否定从前文可能推导出的绝对结论，属于部分否定。

例句

高ければ質がいいわけではない。

并不是贵就质量好。
N2
～に違いない一定是……
＋
接续

普通形 + に違いない

理解重点

表示说话人有较强把握的推断。

例句

彼は事情を知っているに違いない。

他一定知道情况。
N2
～に応じて根据……
＋
接续

名词 + に応じて

理解重点

前项发生变化时，后项也相应调整。

例句

能力に応じて課題を選びます。

根据能力选择课题。
N2
～一方で另一方面……
＋
接续

普通形 + 一方で

理解重点

并列同一对象的两个对照面。

例句

便利な一方で、情報管理が難しい。

方便的另一方面，信息管理很困难。
N2
～にすぎない只不过是……
＋
接续

普通形 / 名词 + にすぎない

理解重点

把事物限定在较低程度，常用于冷静评价。

例句

これは一つの例にすぎません。

这只不过是一个例子。
N1
～を踏まえて基于、考虑到……
＋
接续

名词 + を踏まえて

理解重点

常用于报告、提案和正式决策。

例句

調査結果を踏まえて改善策を考える。

基于调查结果思考改善措施。
N1
～を余儀なくされる被迫……
＋
接续

名词 + を余儀なくされる

理解重点

书面表达，强调受外部原因影响而没有选择。

例句

大雨で計画の変更を余儀なくされた。

因大雨被迫改变计划。
N1
～ならでは只有……才有的
＋
接续

名词 + ならでは

理解重点

强调某人或某物独有的优点和特征。

例句

京都ならではの景色を楽しんだ。

欣赏了京都独有的景色。
N1
～にかたくない不难……
＋
接续

动词辞书形 + にかたくない

理解重点

常与「想像する」「理解する」搭配，书面语。

例句

彼の苦労は想像するにかたくない。

不难想象他的辛苦。
N1
～を皮切りに以……为开端
＋
接续

名词 + を皮切りに

理解重点

表示一连串同类事件从前项开始。

例句

東京公演を皮切りに全国を回る。

以东京公演为开端进行全国巡演。
